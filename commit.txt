  Error Types Handled:

  1. Empty fields - User-friendly validation messages
  2. Invalid credentials - Clear message about wrong username/email or password
  3. Account deactivated - Specific message for banned/inactive accounts
  4. Role mismatch - Prevents managers logging into staff panel and vice versa
  5. Too many attempts - Laravel's built-in throttling with lockout messages

  All errors are displayed using the existing iziToast notification system at the
